Introduction

  • The project is a seat allotment software designed to automate the process of allocating seats in educational institutions.
  • The software simplifies the task of seat allocation by using data provided by the institute authority and the system can allocate seats based on various criteria such as merit, reservation, and availability.
  • The software aims to streamline the seat allocation process and reduce the workload of administrators, making the process more efficient and accurate.
  • The software can help to improve the credibility of the institution or organization.

Objectives

  • To develop a software application to allot seats based on seat-matrix by processing the choice data for various counselling rounds of UG enginerring (1st Year) programs.
  • To generate various reports like Institute wise cutoff, Program wise Cutoff, Most Pre- ferred Branch, Waiting List Report, etc.

Feasibility Study

  • Technological Feasibility: It is technically feasible to use Python to create the software logic and SQL to administer the database.
  • Economic Viability: Python and SQL are a cost-effective combination for creating and deploying the seat allocation program.
  • Operational Suitability: The software should be scalable and capable of handling high data volumes.
  • Legal Liability: The seat distribution program must abide by laws governing data security, privacy, and accessibility.
